Overview

Serial Monitor is a desktop application built with Electron for embedded developers. It provides real-time visualization and analysis of sensor data streamed over USB-UART serial connections.

Supported Runtimes

  • Python — MicroPython and CircuitPython devices. Best for rapid sensor scripting and prototyping.
  • C / C++ — Arduino, ESP-IDF, and bare-metal UART firmware. Ideal for production and low-level control.

Supported Protocols

  • I2C — SHT40, BME680, STS30, STTS751, LIS3DH, VEML7700, TLV493D, VL53L0X, LTR390, VCNL4040, SEN66, Weather Shield
  • RS232 — Wind Sensor (direction and speed)
  • RS485 — MD02 (temperature and humidity)
  • ADC — Rain Gauge (analog-to-digital rainfall measurement)
  • Analog — Hall Sensor (magnetic field), IR Sensor (infrared)
  • GPIO — Blinky (LED), Buzzer, Relay control
  • Weather — Unified dashboard combining wind, rain, temperature, humidity, pressure, and light

Visualization Features

  • Thermometer — Animated liquid-fill gauge for temperature (°C and °F for STTS751)
  • Humidity Wave — Dynamic SVG wave animation that rises with humidity level
  • Pressure Gauge — Arc-based pressure bar (hPa)
  • Light Meter — Sun/moon SVG animation based on ambient lux
  • 3D Accelerometer — Rotating cube for LIS3DH X/Y/Z data
  • Magnetic Vector — TLV493D 3-axis vector visualization
  • Wind Flow — Animated particle system showing wind direction and speed
  • SEN66 Air Quality — PM1.0, PM2.5, PM4, PM10, VOC, NOx, CO2 cards
  • UV Index — Sun glow with color-coded UV bar
  • ToF Distance — Person-to-sensor distance animation (VL53L0X)

Serial Communication

  • Configurable baud rate (default: 115200)
  • Auto-detection of USB-UART bridge on connect
  • Live UART log output with color-coded messages
  • Send custom commands: SET_DEVICE_ID, SET_INTERVAL, GET_INTERVAL
  • Automatic cable-unplug detection and reconnection handling

Keyboard Shortcuts

  • Enter — Quick-connect to the selected runtime
  • Esc — Close this documentation panel
