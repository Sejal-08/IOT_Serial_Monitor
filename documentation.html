Serial Monitor Docs

Application Documentation

Overview

The Serial Monitor is a powerful tool designed for embedded developers to visualize and analyze real-time data from various sensors and protocols.

Supported Runtimes

  • Python: Optimized for MicroPython and CircuitPython devices. Ideal for rapid sensor scripting.
  • C / C++: Perfect for Arduino, ESP-IDF, and bare-metal UART applications.

Supported Protocols

The application supports a wide range of protocols including:

  • I2C: SHT40, BME680, STS30, LIS3DH, TLV493D, and more.
  • ADC: Rain Gauge and other analog-to-digital sensors.
  • RS232: Professional wind sensors and serial devices.
  • GPIO: Control for Blinky, Buzzers, and Relays.
  • Weather: Unified dashboard for combined weather parameters.

Features

  • Real-time Visualization: Dynamic gauges, thermometers, and wave animations for sensor data.
  • UART Logs: Live logging of serial communication with filtering capabilities.
  • Theme Support: Seamless switching between Light and Dark modes.
  • Auto-Detection: Automatic USB-UART bridge detection on connection.

Quick Tips

Press Enter on the login page to quickly connect to your selected runtime.

Use the Refresh button in the dashboard to reset the current session without disconnecting.
